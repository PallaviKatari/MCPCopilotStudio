# MCP Products API (Copilot Studio Ready)

This project exposes a simple MCP (Model Context Protocol) server that provides product data from a public API.

It is designed to be deployed on **Vercel** and connected to **Copilot Studio**.

---

## 🚀 Features

- MCP-compatible tool server
- Streamable HTTP endpoint (`/mcp`)
- Product data tool (`getProducts`)
- Ready for Copilot Studio integration
- Serverless deployment (Vercel)
